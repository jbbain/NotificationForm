READ ME

In order to run php index file, you must download XAMPP and run Apache.

Step 1
Go to the Apache Friends website and download XAMPP for Windows. 
For the easiest install, download the Basic Package's "self-extracting RAR archive." 
Wait for the download to finish and open it to begin installing XAMPP. 
Click the "Install" button to start the file extraction. 
When the Command Prompt screen appears, press the "Enter" key at every question to accept default settings.



Step 2
Start the XAMPP program. 
When started, XAMPP loads itself into your icon tray. 
The icon is orange with a white bone-like shape in its center. 
Single-click the icon to expand the Control Panel. 
Click on the "Start" button next to "Apache" to start your Apache Web server. 
When Apache is running, the word "Running" will appear next to it, highlighted in green. 


Step 3
Place Notification Form project in the "HTDocs" folder located under the "XAMMP" folder on your C: drive. 
The file path is "C:\xampp\htdocs" for your Web server. 
The file path after placing the project should be "C:\xampp\htdocs\NotificationForm"
Open up any Web browser on your desktop and enter "localhost" into the address box. 
The browser will open a list of files stored under the "HTDocs" folder on your computer.
Click on the link to the index.php file and open it to run the script.
